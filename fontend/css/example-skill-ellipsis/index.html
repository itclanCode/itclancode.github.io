单行溢出省略号显示-固定宽度是必须的-文本内容超出部分溢出隐藏,文本内容超出部分溢出隐藏,文本内容超出部分溢出隐藏,文本内容超出部分溢出隐藏,文本内容超出部分溢出隐藏
宽度-width:
内容溢出-overflow:
文本溢出-text-overflow:
段落文本换行-white-space:

实现代码如下


 

 



.single-ellpsis-wrap {
  width: 500px; // 注意宽度是必须的
  overflow: hidden; // 超出部分隐藏
  text-overflow: ellipsis; // 超出部分,用...省略号代替
  white-space: nowrap; // 不换行
}
